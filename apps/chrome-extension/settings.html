Agently Chrome Extension

Settings

Documentation for the current keyboard shortcuts and interaction shortcuts lives here for now.

Bridge Settings

Match this port with the VS Code Agently setting `agently.bridgePort`.

Keyboard Shortcuts

Shift + Left Click

Open the Agently prompt panel near the clicked element and capture DOM context.

Enter

When queue mode is enabled, add the current prompt to the queue from the prompt field.

Shift + Arrow Up

Start voice input while the prompt textarea is focused.

Escape

Stop voice input, or close the drawer and bottom-panel modes.

Toolbar Icon Click

Open Agently as a bottom panel on supported localhost pages.

Notes

Keep the Chrome extension bridge port aligned with the VS Code bridge port.
